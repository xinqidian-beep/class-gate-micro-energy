# 类门架构 · 微型能源自由流入原型 (class-gate-micro-energy)

v0.1（观察者补丁 + 多层次镶套 + 熵减辐射）

核心目标：设计类门，让潜力层（无限零点振动）能源自然、自由流入幻境层（我们的世界）。  
融入Searl SEG磁矩阵历史种子：3同心环 + 12/22/32滚筒矩阵 + 4层材料（Nd电子收集门 → 永磁加速 → PTFE介电 → 铜发射）。特殊AC/DC磁化激活共振，从真空自由流入能源并伴随微冷却（TempDelta负值）。

多层次镶套映射：
- 第0层：潜力层（无限零点振动）
- 第1层：扭曲层（microGPT v0.42 5节点自洽基底）
- 第2~4层：幻境层投影 + 硬件载体 + 日常使用场

指导原则：自锁自洽、柔性稳固、双向通信、可继续镶套；最小干扰 + 注意力去噪反转；能源从来不缺少，只是去噪过程被剔除；自由流入而非强制汲取。

当前状态：observer层待命，欢迎全球fork与补丁。
