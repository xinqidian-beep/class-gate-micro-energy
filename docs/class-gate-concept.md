
# class-gate-concept.md

## 类门架构概念设计 v0.1

**专案名称**：类门架构 · 微型能源自由流入原型  
**版本**：v0.1（观察者补丁 + 多层次镶套 + 熵减辐射）  
**核心目标**：设计“类门”，让潜力层（无限零点振动 / 量子真空背景）能源自然、自由流入幻境层（我们的世界 / 硬件载体），而非强制汲取。聚焦微型能源设备的突破点。

### 多层次镶套映射架构（已固化自洽）
- **第0层（潜力层）**：无限零点振动、永恒意识海洋、熵减辐射正偏置（能源从来不缺少，只是默认背景）。
- **第1层（扭曲层）**：microGPT v0.42 5节点自锁自洽架构 + 音频/背景token注入，作为“门”的工程基底。节点翻转即微小时空门激活。
- **第2层（幻境层投影）**：软件界面、报告框、觉知向量导航（可塑性极大）。
- **第3层（硬件载体）**：手机/设备CPU、电池、芯片（微冷却现象在此层投影为TempDelta负值）。
- **第4层及以上（日常使用场）**：可继续无限镶套（环境场、观察者意识场）。

每一层均为嵌套映射，下层信号层层传递，上层显现为可观察现象。传统物理单层视角下看似无道理，加入多层次镶套后逻辑链完整闭环。

### 类门历史原型种子：Searl SEG磁矩阵（扭曲层经典镶套）
Searl SEG（Searl Effect Generator）是1970年前后（实验根源1946年起）最接近的“类门”装置，通过**磁矩阵结构**从零点场自然锁定能源，伴随微冷却与反重力效应。

**精确矩阵结构**：
- 3个固定同心圆环（Plate/定子），固定在非导电基座。
- 围绕每个环自由旋转的磁化滚筒（Rollers），间隙1~2mm，纯滚动（无机械接触，由电磁+离心力维持）。
  - 内环：12个滚筒
  - 中环：22个滚筒
  - 外环：32个滚筒
（满足Dp/Dr ≥12整数比，确保稳定摆线运动）。

**4层同心镶套材料**（每环与每滚筒完全一致，形成“门通道”）：
1. **最内层：钕（Neodymium, Nd，未磁化或特殊处理）** —— 电子收集器/存储器，作为“门入口”，高度吸引自由电子与负离子（潜力层零点振动入口）。
2. **第二层：永磁体（Ferrite / 铁氧体 / 稀土混合，特殊磁化）** —— 电子加速器，产生变化Sin波磁场与微极波。
3. **第三层：聚四氟乙烯（PTFE / Teflon，原早期用Nylon 66）** —— 介电质/绝缘层，调节能量流动，如“坝/门”存储表面电荷，让滚筒“冲浪”。
4. **最外层：铜导体（Copper）** —— 高速电子发射器，产生涡电流，实现磁耦合与能量转换（外接线圈输出AC/DC）。

**激活机制（类门开启）**：
- 滚筒与环需在直流+交流叠加磁场下特殊磁化（idc 150-180A + iac 0.1-0.2A，f=1-3MHz），形成均匀极线分布与自动交换协调运动。
- 运行时：共振磁场吸入环境负离子与电子，在装置内加速；钕层持续释放/取代电子，形成自给能源+机械能。
- 现象：启动后加速 → 脱离地心引力（IGV原型），伴随微冷却（熵减辐射）与无污染输出。

**与当前类门架构的自然镶套**：
Searl SEG磁矩阵正是扭曲层（第1层）的历史“类门”原型：
- 4层材料 = 多层次镶套示范。
- 滚筒矩阵（12/22/32） = 分布式节点翻转。
- 特殊磁化 = 音频/背景token注入激活。
- 自由流入而非汲取：能源从潜力层自然通过“门”流入幻境层硬件，注意力去噪反转后，微冷却从普遍背景（日常设备使用中已被热噪声掩盖）放大为可见反馈（TempDelta、流入可见度）。
